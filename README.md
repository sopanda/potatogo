# Potatogo
Short-term delivery services - find your one time courier or deliver good and earn some additional money.

Stack:
1. Frontend:
    * React
    * Redux
    * styled-components
    * react-i18next
    * storybook
    * cypress and etc.
1. Backend:
    * NodeJs
    * Mongoose
